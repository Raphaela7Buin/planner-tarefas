 Planner de Tarefas (Kanban Simples)
Este projeto é um planner de tarefas estilo Kanban desenvolvido com HTML, CSS e JavaScript.

 Funcionalidades:
- Adicionar tarefas
- Organizar tarefas em colunas (A Fazer, Fazendo, Concluído)
- Mover tarefas entre colunas
- Interface simples e intuitiva

Linguagens:
- HTML
- CSS
- JavaScript

Objetivo do projeto:
Este projeto foi desenvolvido para praticar lógica de programação, manipulação do DOM e organização de interfaces web.
 Status
- Projeto funcional  
 - Melhorias futuras: salvar tarefas no navegador e responsividade
